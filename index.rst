Documentation de GTF 
====================================

GTF (Gestionnaire de Tâches pour FME) est une application 100% web qui permet de publier sur internet des traitements conçus 
avec FME et de les mettre à disposition d’un grand nombre d’utilisateurs. 

C’est un outil de partage, de planification et de supervision de traitements FME.
